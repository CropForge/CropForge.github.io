Managing Pedigree Nursery Information with IRIS

Introduction:

Objectives: At the end of this chapter, the user will be able to:

  1. Generate the F3 and F4 nursery lists.
  2. Generate germplasm derivatives interactively;

Tutorial Steps:

In this exercises, we will form the F3 nursery list by selecting four plants from each plot in the F2 nursery. On the other hand, we will generate the F4 nursery by bulking the seeds from selected rows in the F3 nursery.

Generating the F3 Nursery List:

  1. Start SETGEN from the Launcher (see ICIS TDM 5.2).
  2. Select the U04WSF2 list from the List Selector and click <List><Browse> (see ICIS TDM 5.5.2).
  3. Click on the <List><New> menu.
  4. Make a new list called U05DSF3 with ‘Upland F3 Nursery 2005 Dry Season’ as the title of the list.


    Figure 1. After opening a list in the Browse window and creating a new list in the Edit window.

  5. Select all the entries from the browse window. This is done by using the mouse to highlight the entries or by pointing the mouse on the Tag column header then right click the mouse and choose Tag All. After selecting the entries, notice the letter ‘X’ appeared in the ‘Tag’ column. This indicates that those entries tagged with ‘X’ have been selected.


    Figure 2. Selecting entries in the Browse window for advancing a line.

  6. Click the ‘-‘ button. The Advance a Line One Generation window will appear.
  7. Specify the necessary information needed for advancing the F2 lines. For this exercise, we will pick the DER 205 DSP SINGLE PLANT SELECTION for the Derivative Breeder Method, and we will put ‘4’ at Plant Selected under the Naming Conversion section.
  8. Supply also the dates and location under the Germplasm Date and Location section. (see ICIS TDM 5.5.3) For this exercise, type 2005-01-01 for the date, and select IRRI-INTERNATIONAL RICE RESEARCH INSTITUTE, LOS BANOS for the location.

    To change the location click on the <Change> button to make the Select Location window appears, and choose PHILIPPINES as Country, and select IRRI-INTERNATIONAL RICE RESEARCH INSTITUTE, LOS BANOS found in the list box under the location section.

  9. After specifying the information in the Advance a Line One Generation window, click the <Ok to All> button.


    Figure 3. The Advance a Line One Generation window


    Figure 4. Return result after advancing the selected entries.

  10. Notice that the ‘Entry Code’ column has been assigned with a default value of carrying previous entry code suffixed by an increasing number. To change the values to this column, point the mouse cursor at the ‘Entry Code’ header column and right click the mouse. A small menu box will appear with some options you can select from to fill in the ‘Entry Code’ column. For this exercise choose ‘Fill with Sequence’.
  11. The Naming Convention window will appear. Type ‘P’ in the Start Constant section, type ‘1’ in the Next Integer section and F3 in the End Constant section then click on the Include leading zeros check box and type ‘3’ on the Numeric Field Width section then click on the ‘Close’ button. Notice that the ‘Entry Code’ column has been filled up with sequential numbers.


    Figure 5. Naming Convention window.


    Figure 6. Return result after filling up the Entry Code with sequential numbers.

  12. Click on the <List><Save> to save the new list.
  13. You could export the list into an XLS fieldbook by clicking <List><Export As><XLS Fieldbook>.
  14. Click on the <List><Close List><All> to clear the Edit and Browse windows.

Generating the F4 Nursery List

  1. Select the F3 Nursery List (U05DSF3) and open it on the browse window.
  2. Create a new list called U05WSF4 with ‘Upland F4 Nursery 2005 Wet Season’ as the title of this list.
  3. Randomly select about 50 to 70 entries in the browse window. These represent the entries you choose to advance by field inspection and data collection on the F3 nursery.
  4. Click on the ‘-‘ button to advance the selected entries in the browse window by one generation.
  5. Pick the ‘DRV 207DRB RANDOMBULK SF’ under the Derivative Breeding Method section. Check if the dates and locations under the Germplasm Date and Location still have the suitable information.
  6. Click the <Ok to All> button. A small confirmation box will appeared saying “No naming convention specified. Is the derivative name same as its source?”. Click Yes.
  7. An information window displays notifying the user that a germplasm with the same name was found.
  8. Select ‘Add another germplasm with this name’ to add the same germplasm name but generate a different GID to the list.
  9. Click on ‘Ok to All’. This will generate new GIDs with the same germplasm names for all of the items in the list.
  10. Repeat steps 10-12 of the Generating the F3 nursery list exercise to generate and save the F4 nursery list.
