Managing Seed Stock - Deposit

Introduction:

Objectives: At the end of this chapter, the user will be able to:

  1. Create lot locations for the F5 list.
  2. Deposit F5 seeds for stocking.

Tutorial Steps:

Creating lot location for the F5 list

  1. Inside SETGEN, select on the U06DSF5 list (double click).
  2. Tag all entries on the edit window.
  3. Right-click on the white space and select <View Inventory> on the pop-up menu.
  4. The Inventory Tracker window will display.
  5. Right-click on the inventory details section of the Inventory Tracker window and select <Add Lot> from the pop-up menu. A message will display prompting the user to select GID, location and scale.


  6. On the germplasm section, click on the <tag> column header to tag all records.
  7. On the inventory filters section, click on the elipses button to view the location window.


  1. Select <Short Term> on the Collection drop-down list then select <ST, Tray No. 998> on the Tray drop-down list. Click on the <OK> button to close the window and update the location field on the inventory details section.
  2. Select <Packet> on the Inventory Units drop-down list.
  3. On the Comments field, type “lot created” and type the current date.


  4. Click on the <Save> button on the toolbar to update the database. The new lot locations are displayed on the Inventory Details section.


Depositing seed stock for the F5 list

  1. Tag all the items in the Inventory Details section.
  2. Right-click on the white space and select <Deposit> on the pop-up menu. The Transaction window will display on the screen.
  3. Encode the number of units to deposit, commitment date of the deposit and any comment on the deposit being made, on the space provide for each item.


  4. If the values for all items are the same, click on the column headers of <Units to Deposit>, <Commitment Date> and <Comments>, then a pop-up menu will be displayed.


  5. Click on the enabled sub-menu item to display a small window where data can be entered.








  6. When you want to assign a donor for this transaction, select an item from the Donor drop-down list.


  7. The system will show a confirmation message on sending an email request to the administrator.


  8. Click on <No> to continue without sending an email message to the administrator.
  9. Click on <Ok> to save changes to the database. The Transaction Window closes and the system goes back to the Inventory Tracker window.
