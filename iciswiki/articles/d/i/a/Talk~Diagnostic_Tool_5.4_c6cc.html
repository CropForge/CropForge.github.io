Q&A

Q: When running the DLL test, what does it mean when you encounter an error message like this?

A: As of version 5.4 of ICIS applications and schema, a new column (USERID) was added to the STUDY table. The error message above means that your STUDY table must have a USERID column in order for the ICIS DLL function DMS_GetStudy to function.

Suggestions/comments for next release

  • Open generated RESULTS text file at the end of the test (c/o Vivay Salazar).
  • Support for User DSNs (c/o Jesper Norgaard).
  • Support for different backends (other than MS Access. i.e.: MySQL, Postgres, Oracle) for Database Test.
