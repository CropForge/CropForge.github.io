Review of DMS and GEMS 6.0 schema and implementation of ICIS Data Source use cases for Molecular Breeding

Here you will find all information required for the review and implementation. In most cases these are links to existing wiki pages that need to be read and in some cases updated during the review. By only maintain links here these pages can be used as a resource after the review has finished.

Scope of Review

The review is restricted to following areas:

  1. Development of query, visualisation and analysis user requirements for genotyping studies for molecular breeding
  2. Formalized query use cases for these user requirements
  3. Review of GEMS version 6.0 schema in respect to these use cases
  4. Implementation of use cases in the GCP ICIS Data Source
  5. Implementation of GUI in Zeus and GenoMedium for these use cases
    1. GenoMedium query wizards or Zeus 'forms' for gathering user parameters for these use cases
    2. Development of new visualisation or integration of existing visualisation for genotyping studies
Timetable

Please note the dates are deadlines for completion, the work may completed before then and the timetable updated.

April 14
  • Read and understand relevant and existing ICIS wiki and Pantheon documents (see Supporting documents)
April 30
  • Update list of User requirements for molecular breeding informatics
May 31
  • Update Proposed GEMS 6.0 schema based on user requirement, if necessary
  • Complete Proposed extension to DMS (DMS 6.0 schema) based on user requirement, if necessary
  • Formalise query use case in GCP platform
  • Collatate real data that exemplifies the user requirements
June31
  • Populate the proposed GEMS 6.0 schema and associated (proposed) DMS tables with these example data
  • Develop and performance test the SQL queries required for the use cases using these data
  • Implementation of GUI in Zeus* and GenoMedium for these use cases
July 31
  • Implementation of GCP use cases for ICIS
  • final work on these at the mini-ICIS meeting
July - T.B.A.
  • Mini-ICIS meeting in Mexico
