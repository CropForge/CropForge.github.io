User Input Fields

Field Name Description
Crop Year Show entry(s) according to year and growing season of the crop.
Species Show entry(s) according to species.
Selection Type Show entry(s) according to initial seed increase or replanting.
Total Total number of entry(s) planted according to location.
Location Show entry(s) according to location. (i.e. field, nursery, laboratory or not selected).
Show Show entries according to selected or deleted entries.
DEL Check to delete entry.
PM SRC Planting Materials Source. Indicates whether the data is for replanting or harvest.
Batch ID Identification code assigned to a batch.
Temp ID Identification code assigned to a sample.

Variety Name Local/vernacular name of the sample.

Replanting Information

Field Name Description
Crop Year Year and growing season of the crop.
Plot No Number assigned to each plot.
Remnants - Status Either insufficient seed remnants or none at all.

Use Case Definition

Use Case Name 2.1.1 Finalize Selected Samples for Planting
Use Case Definition This use case allows the actor to finalize the seed list or the selected plants for planting. This allows him/ her to delete manually a sample from the list.
User Contacts
Actors Genebank Manager (GM)
Location Genebank
Priority 1
Typical Course of Events
Actor Action System Response
Step1: This use case is initiated when the actor selectes a crop season that needs to be updated Step2: The system retrieves the list.
Step3: GM deletes a specific sample from the list Step4: System marks the sample in the list as deleted (listdata)
Step5: GM requests for a proof list Step6: System returns the prooflist
Step7: Use case concludes when the GM receives the report
Assumption/s Prelininary samples selected for planting is already existing
Pre-condition/s Seed list marked as preliminary
Post-condition/s Final seed list is ready for seed preparation. Samples deleted from the list is marked/ flagged accordingly in the database.
Primary Pathway/s Finalize seeds for planting
Alternative Pathway/s Delete list through ICIS SetGen
Exception Pathway/s
