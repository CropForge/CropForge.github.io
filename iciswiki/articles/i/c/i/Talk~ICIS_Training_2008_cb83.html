Course Description

The International Crop Information System (ICIS) is a database system for the management and integration of crop research data, which includes specialist user interfaces for diverse disciplines. This course is primarily for plant breeders who want to use ICIS to manage breeding program data, but will be useful for crop scientists who manage data for research on genetic resources or germplasm evaluation.

The course consists of lectures and hands-on practical exercises using ICIS with rice data. It aims to introduce participants to the various ICIS components, and the available tools and modules within them. Topics included are tools and modules in the following components:

  • Genealogy Management System (GMS)
    • Name Searching
    • Generating/Importing Nursery Lists
    • Prescribing Crosses or Advancing Lines
    • Editing Characteristics
  • Data Management System (DMS)
    • Using Workbook
    • Formulating Data Retrieving Queries
  • Inventory Management System (IMS)
    • Tracking of Seed Stocks
    • Depositing/Reserving of Stocks
    • Labels and Bar coding
