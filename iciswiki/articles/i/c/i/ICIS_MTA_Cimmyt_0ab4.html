Strategies for Creating the List of Entries

Wheat

Strategy A - approved strategy
  1. Generate the Nursery list in SetGen. (Add another tool in SetGen where other attributes of a list can be entered: Occurence, Trial ID, Cycle=>which can be year of LISTDATE, Program, Cooperator etc. Also, add the facility in GMS Search to search by SID,CID, selection history, pedigree etc))
  2. Make a trigger to create the nursery list in IWIS system based on the information provided in step 1.
  3. Follow the usual step to fill up the sMTA Workbook template by retrieving the nursery list from SetGen and all necessary information using the ICIS Workbook
For remote CIMMYT stations
  1. Remote station site (e.g. Turkey) will prepare the list
  2. CIMMYT central office will create the nursery list
  3. CIMMYT central office will generated the SMTA document


Strategy B
  1. Generate the Nursery list in IWIS
  2. Add another type of report in IWIS that will populate the sMTA Workbook template OR a trigger to automatically create a nursery list in a local ICIS database
  • Create a report in the current IWIS system that will export the list of entries in the Workbook Excel file with one of these options
    • Only the list of entries with GID and the recipient will be exported to the Workbook
    • All necessary information will be exported in the Workbook
Notes
  • The very same tool that generates the fieldbook that is downloadable in the web can be modified to populate the Workbook Template.
  • The possible mapping of data from the IWIN fieldbook with the Workbook template
Trial Name <=> Title
The full name associated with the Cooperator ID <=> RECIPIENT
Entry <=> ENTRYCODE
Cross Name + Selection History <=> DESIGNATION
Trial Abbr <=> CATEGORY
Introduction Number <=> SAMPLE UNIQUE ID
  • Other info that is more likely in IWIS
Recipient Address
Sender
  • Other necessary information in the sMTA template
GID
Species
Country of Origin
MLS Designation Date
  • Develop a trigger that will generate a GMS list in a local ICIS database as new nursery trial is created in IWIS. Then follow the usual routine of filling up the sMTA Workbook template
Issues
  • Should there be one GMS List for each nursery?
  • Should there be different list per nursery/cooperator? The GMS list name can be the name of the SMTA reference number itself.


Notes:
Both strategies will still require the information about the MLS Designation to be in GMS and to use ICIS Workbook to fill up the MLS Ancestor sheet of the template and to generate the sMTA document

Maize

Notes:

  • There are two ways a list of requested seeds is processed. One is through the Genebank system and the other one is through breeder's field book
  • If a request contains seeds both from genebank and breeders, two kinds of list will be in the request.
  • We can assume that all genebank materials are under SMTA.
  • Breeders' seeds can be SMTA-Under development if there are accessions and released lines within its pedigree, otherwise it might have a different type of MTA.


Strategy for Genebank
  1. Create a macro in the Workbook to populate the SMTA template from the ASCII text file generated by the Genebank system
  2. Retrieve GID from IMIS (e.i. the maize implementation of ICIS) based on Accession Number. It is assumed that one GID is assigned to an Accession number. A command in ICIS Workbook that can retrieve the GID based on Accession number should be developed.
  3. Use the ICIS Workbook to fill up the rest of the information in the SMTA template.
  4. Use the ICIS SetGen to enter the list of newly imported seeds.
  5. Create a tool in Genebank system that will get the newly imported list from IMIS.


Strategy for Nursery Trials/Breeders' Seeds

Strategy A

  1. The nursery list or request list will be created in IMIS local database using SetGen
  2. Follow the usual step to fill up the sMTA Workbook template by retrieving the nursery list from SetGen and all necessary information using the ICIS Workbook
  3. Develop a macro in ICIS Workbook that will populate the Maize Fieldbook from a SetGen list
StockID => can be stored in IMIS as Source or Entry Code
Name => the preferred name in IMIS
Pedigree => Cross expansion in IMIS

Strategy B

  1. Create the list in Maize Fieldbook
  2. Use GMS Input to transfer list from Field book to GMS
  3. Follow the usual step in filling up the SMTA Workbook template
Note:
  • What is be the Unique Sample ID in nursery trial/breeders' seed? Is that the Stock ID? Is Stock ID unique across maize lines?
  • It is being investigated if CropFinder can be used to give the descriptive and passport information for the entry of a requested list.
  • Use ICIS Web page for germplasm information and follow the IRIS SMTA web portal. Just modify the code that shows the attributes and descriptive information about an entry.
