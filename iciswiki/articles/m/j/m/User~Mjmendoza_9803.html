Job Profile:

I joined IRRI last June 2004 and started working on developing a prototype LIMS-based program using JSP/Java Servlets and MySQL. I then managed and continue to administer and support the network operation and security of IRRI’s High Performance Computing (HPC) cluster/grid with 16-nodes parallel computer running Linux. At the start of 2005, I was able to commission a web site for collaborative management of software development (CropForge) and document editing (CropWiki). Currently, I continue to provide system administration and installation support to the core Linux servers of CRIL including integration to IRRI’s network, implementation of backup strategies for critical data, and employment of local network security.
