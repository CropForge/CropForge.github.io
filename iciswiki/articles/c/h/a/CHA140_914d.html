Query Seed Request

Overview

The Query Seed Request form allows the user to search for seed requests (for characterization), and samples included in seed requests. The user may request for a prooflist of the data.

Use Case Definition

Use case name

1.4.0 Query Seed Request

Use case description

This use case involves searching for seed requests and viewing of samples included in a particular seed request (for characterization)
User contacts n/a
Actors Genebank technician
Locations Genebank
Priority 1
Typical Course of Events:
Actor Action System Response
Step 1. Use case is initiated when the Actor specifies the criteria for searching of the seed request(s) Step 2. System retrieves all seed requests that match the criteria provided by the Actor
Step 3. Actor requests for prooflist of samples included in a seed request Step 4. System produces the prooflist
Step 5. Use case is concluded when the Actor receives confirmation
Assumption/s View only
Precondition/s n/a
Post condition/s n/a
Primary pathway Query seed requests and samples included in the requests
Alternate pathway/s n/a
Exception pathway/s n/a
