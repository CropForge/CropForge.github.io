Requirements

Steps used for evalution

  1. Is released on LGPL or equivalent? No then fail, if Yes go to 2.
  2. Does database schema work on MySQL at least? No then fail, if Yes go to 3.
  3. Is Java used to connect to the database? No then go 5, if Yes go to 4.
  4. Test Java code
  5. Test database schema on MySQL

Conclusions (so far)

At the moment it has not yet been possible to find a system or schema that fits the above criteria for use with ICIS
