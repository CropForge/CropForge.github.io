Overview

The form allows the user to print labels. The user is also allowed to select the position where he wants start printing the label in a 3x9 A4 sheet.

Use Case Definition

Use Case Name 4.1 Label Printing
Use Case Definition This is the generic way of printing barcode labels. The users can print either by accession numbers OR by location (tray numbers).
User Contacts n/a
Actors Genebank technicians
Location Genebank
Priority 1
Typical Course of Events
Actor Action System Response
Step 1: The user supplies the accession number and the packing material where the labels need to be stuck. Step 2: The system processes the information and generates the seedlist and creates the labels.
Step 3: The user views and prints the labels. Step 4: System prints the report.
Step 5: Use case is concluded when the actor receives the process confirmation.
Assumption/s The germplasm already exists in the inventory.
Pre-condition/s Create a seedlist and barcode labels printed
Post-condition/s
Primary Pathway/s
Alternative Pathway/s
Exception Pathway/s n/a
Status 100% Completed
